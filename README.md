# Sagedash — Expert parenting coach for teen communication
**Tagline:** Bond Closer. Rise Together.

Sagedash gives parents three short, loving lines they can say right now—tailored to their teen.

## Stack
- React + TypeScript + Vite + Tailwind (PWA)
- Hosting: Netlify
- Firebase Auth/Firestore, Stripe, AI endpoint

## Quickstart
```bash
npm i
npm run dev
